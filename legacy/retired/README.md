# Retired systems — do not install

Everything in this directory is preserved only as historical reference.

These files implement the abandoned generic campaign, tornado/D.E.S. crisis,
advanced progression, or old Time Machine/future-apartment experiments. They
are intentionally absent from the active Studio place and must never be copied
or broadly synchronized over the current family-task game.

`garage_layout_fix.luau` is also retired: it is the old garage/storm-equipment
installer and is not the source of the current detached-garage environment.

In particular, the retired files named `CooperGame.server.luau`,
`CooperGame.client.luau`, and `CooperGameConfig.module.luau` are **not** the
source of the live Studio instances with those names. See
`MILESTONE0_BASELINE.md` for the canonical mapping.

Any future horror system must be implemented against the current
`CooperFamilyTask*` foundation and four-cutout NPC controller. Do not revive
these scripts as a shortcut.
